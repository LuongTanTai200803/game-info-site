# Vì sao chọn GitHub thay Database?

- Dữ liệu ít.
- Muốn version control.
- Không cần CRUD API.

---

# Vì sao dùng Supabase?

- Chỉ cần Storage.
- Không cần Database.
- Có SDK chạy trên client.

---

# Vì sao dùng Decap CMS?

- Không tự viết Admin.
- Hỗ trợ Markdown.
- Commit trực tiếp GitHub.

---

# Vì sao dùng Netlify?

- Identity.
- Git Gateway.

---

# Vì sao Website dùng Vercel?

- Tối ưu Next.js.
- Build nhanh.
